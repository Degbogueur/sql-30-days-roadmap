# SQL 30 Days Roadmap

Un défi personnel de **30 jours pour (ré)apprendre SQL**, des bases jusqu’aux concepts avancés.

Chaque jour comprend des exercices et leurs solutions sur un concept précis.

Les solutions sont fournies et toutes les requêtes sont testées sur **SQL Server** (syntaxe majoritairement standard SQL).

**Objectif** : pratiquer SQL de manière progressive, concrète et structurée.
